# Codex Notes

## Project

Finebro is a React, TypeScript, Vite, and Tauri finance tracker. Keep the app local-first and responsive across desktop and mobile.

## Working Conventions

- Prefer small, intentional commits.
- Do not commit local secrets or machine-specific configuration.
- Add or update verification steps when introducing application code.
- Keep finance calculations in plain TypeScript modules so they remain portable across UI, native, and sync layers.
